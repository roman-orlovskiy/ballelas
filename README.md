# BALLELAS - Локальная разработка

## Быстрый старт

### Установка зависимостей
```bash
npm install
```

### Запуск сервера разработки
```bash
npm run dev
```

Сервер автоматически запустится на `http://localhost:8000` и откроется в браузере.

### Сборка для продакшена
```bash
npm run build
```

Собранные файлы будут в папке `dist`.

### Превью продакшен сборки
```bash
npm run preview
```

## Остановка сервера

Нажмите `Ctrl+C` в терминале, где запущен сервер.
